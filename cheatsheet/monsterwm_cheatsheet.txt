monsterwm Cheat Sheet
=====================

Start program
-------------

Mod1-Shift-Return : Start terminal
Mod4-v            : Start dmenu

Switch desktop
--------------

Mod1-F1 : focus desktop 1
Mod1-F2 : focus desktop 2
Mod1-F3 : focus desktop 3
Mod1-F4 : focus desktop 4

Mod1-Ctrl-h : focus the previous desktop
Mod1-Ctrl-l : focus the next desktop

Mod1-Shift-h : focus the previous desktop that has windows open
Mod1-Shift-l : focus the next desktop that has windows open

Mod1-Tab : toggles to the last selected desktop

Set layout
----------

Mod1-Shift-b : sets bottom stack layout
Mod1-Shift-f : sets float layout
Mod1-Shift-g : sets grid layout
Mod1-Shift-m : sets monocle layout
Mod1-Shift-t : sets tiled layout

Operate window
--------------

Mod1-j         : focus next window
Mod1-k         : focus previous window
Mod1-Backspace : focus the window with an urgent hint

Mod1-l : increase master area size
Mod1-h : decrease master area size

Mod1-o : shrink the size of the first stack window
Mod1-p : grow the size of the first stack window

Mod1-Shift-F1 : move focused window to desktop 1
Mod1-Shift-F2 : move focused window to desktop 2
Mod1-Shift-F3 : move focused window to desktop 3
Mod1-Shift-F4 : move focused window to desktop 4

Mod1-Return  : swaps the focused window to/from master area
Mod1-Shift-j : move the focused window down the stack
Mod1-Shift-k : move the focused window up the stack

Mod4-{j,k,h,l}       : move the current window to the corresponding direction
Mod4-Shift-{j,k,h,l} : resize the current window to the corresponding direction

Mod1-Shift-c : close focused window

Toggle panel
------------

Mod1-b : toggles the panel on and off

Exit monsterwm
--------------

Mod1-Shift-r : quit with exit value 0
Mod1-Shift-q : quit with exit value 1
